Land Cover Classification Using Machine Learning and Remote Sensing

This research investigated the application of machine learning techniques to land cover classification using multi-source remote sensing data. A land cover map for the Republic of Macedonia was produced by combining optical and radar data from the Copernicus Sentinel-1 and Sentinel-2 satellites with ancillary topographic information. Support Vector Machine and Random Forest algorithms were applied to classify land cover, achieving high performance and demonstrating the potential of machine learning for large-scale environmental monitoring. The study highlighted the value of integrating open satellite data and advanced analytics for accurate, cost-effective, and timely land mapping.
